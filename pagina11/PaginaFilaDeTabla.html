Â¿Para que sirve una fila de tabla?

En HTML, una fila de tabla se representa mediante la etiqueta <tr> (table row). Dentro de una fila, puedes colocar celdas de encabezado (<th>) o celdas de datos (<td>). La fila organiza los datos de una tabla de forma horizontal, mientras que las celdas dentro de ella contienen el contenido.
